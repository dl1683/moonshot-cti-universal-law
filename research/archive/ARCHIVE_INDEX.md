# Research Archive Index

These documents are from Phase 1 of the CTI project (Feb 2026), which explored a universal power-law relationship D(C) = D_inf + k*C^(-alpha) for layer-wise representation quality. That hypothesis was **falsified** (see CTI_PIVOT_STRATEGY.md) and the project pivoted to the kappa_nearest law, which is the current paper.

These files are preserved for historical context only. They do not reflect the current state of the project.
